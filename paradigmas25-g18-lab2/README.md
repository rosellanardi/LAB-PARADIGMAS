# LABORATORIO 2 - PARADIGMAS DE PROGRAMACIÓN

## Dependencias:
Se deberá tener instalado:
- Java Development Kit (JDK).

## Compilación y ejecución:
Se otorga un Makefile para la ejecución de la aplicación. A continuación se listan los comandos y su uso:
- make: compila todos los archivos `.java`.
- make run: ejecuta la aplicación sin la utilización de heurísticas (solo `FeedReaderMain`)
- make run-ne: ejecuta la aplicación utilizando la funcionalidad de heurísticas (`FeedReaderMain -ne`)
- make clean: elimina todos los archivos compilados (`.class`)
